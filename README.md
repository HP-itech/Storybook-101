# Storybook-101

## Getting started 

1.  **Install the dependencies.**

   Install the necessary dependencies.

    # Install the dependencies
    yarn
     

1.  **Browse your stories!**
Run `yarn storybook` to see your component's stories at `http://localhost:6006`
